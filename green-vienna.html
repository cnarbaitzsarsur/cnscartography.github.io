Green Vienna interactive map

Web app coded with javascript, css, html and sql.
Project for TU Vienna, 2022.

Green Vienna is a project which analyses and presents, through an interactive map, some of the elements that make this city one of the greenest cities in the world.
At this stage, the extension of the study case is Inner Vienna and the 3 main topics to be considered are trees, parks, and infrastructure.
